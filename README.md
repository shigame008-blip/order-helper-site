# Order Helper

เว็บช่วยสร้างฟอร์มออเดอร์สำหรับแอดมินขายสินค้าออนไลน์

## วิธีใช้

เปิด `index.html` หรือ `order_helper.html` ใน Chrome แล้วใช้งานได้ทันที ไม่ต้องติดตั้งเซิร์ฟเวอร์

## GitHub Pages

ถ้าเอา repo นี้ขึ้น GitHub ให้เปิด GitHub Pages จาก branch หลักและเลือก root folder เว็บจะเปิดจาก `index.html`
